3..0SDRC_MODE
AT91C_SDRC_MODE		Mode
0: Normal Mode.
1: Issue a NOP Command at every access.
2: Issue a All Banks Precharge Command at every access.

3: Issue a Load Mode Register at every access.
4: Issue a Refresh
